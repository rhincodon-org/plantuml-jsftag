# plantuml-jsftag

JSF2.2 Custom Component Code for PlantUML.

## Usage

* add the jar file with plantuml in JavaEE lib dir.
* add xmlns:pu="http://xmlns.jcp.org/jsf/component" and the snippet code in your xhtml file.
```
<pu:plantuml>

  "Bob" --> "Alice"
</pu:plantuml>
```
